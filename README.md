# RA-KG
Knowledge graph applicable to navigation tasks
# Project Overview
A 3D Knowledge Graph visualization project. Includes Neo4j exported data (RA-KG_9room.json) and a Python visualization script.
# Installation
1. Create a virtual environment. We develop this project with Python 3.9.
```
conda create -n RA-KG python=3.9
conda activate RA-KG
```
2.Install the required dependencies using pip:
```
pip install -r requirements.txt
```
